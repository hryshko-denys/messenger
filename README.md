## Tech stack
- HTML
- SCSS
- BEM
- JS (For small screens)

## Instruction

To run project locally:
1. npm i
2. npm start
3. open your browser at http://localhost:3000

If you want to try mobile version:
 - change width of screen
 - reload
 - click on any chats
 - use 'back' button to go back
